# 建立第一個app

```python
from flask import Flask
app = Flask(__name__)

@app.route("/") 
def home():  
    return "Hello"
    
app.run()
```
methods是用來定義這個路由是否執行相關的GET、POST
取得資訊的時候GET
送出資訊的時候POST
更新資訊的時候UPDATE
刪除資訊的時候DELETE

預設為允許GET
